日语学习-零基础入门-02

Posted on 2024-01-13
b站言趣日语小厉

1. AはBです == A是B

は – wa音 — 语气助词

です　– 是 — 表判断

1.1. 我是中国人

わたし　は　中国人　です

1.2. 他是中学生

彼　は　中学生　です

2. AはBでは　ありません == A不是B

は – wa音 — 语气助词

では　ありません　– 不是 — 表判断

2.1. 我不是学生

わたし　は　学生　ではありません

2.2. 他不是学长

彼　は　せんぱい(先輩)　ではありません

注：口语中 では == じゃ

3. AのB == A的B

3.1. 我的书

わたし　の　ほん

3.2. 日语书

日本語　の　ほん

3.3. 京华大学的学生

京華大学　の　学生

注：日本(にほん) 昨日(きのう)

4. AはBですか == A是B吗

か – 终助词 — 表疑问

书写时：

​ 一般疑问 – 使用句号结尾，而不使用问号

​ 强烈疑问 – 使用问号结尾

4.1. 你是学生吗

あなた　は　学生　ですか。

4.2. 她是千鹤老师吗

かのじょ　は　千鹤(ちじる)　先生　ですか

4.3. 一般疑问句的肯定回答

はい/ええ(相对不正式)、そうです 嗯，是的

はい/ええ(相对不正式)、～～です 嗯，是~~(这里的 ~~ 可以替换为问题里的名词，例如学生/千鹤老师/…)

4.4. 一般疑问句的否定回答

いいえ、～～ではありません 不，不是~~(这里的 ~~ 可以替换为问题里的名词，例如学生/千鹤老师/…)

いいえ、ちがいます～～です 不，不是的，是~~(这里的 ~~ 可以替换为正确的名词，例如社员/是白鸟老师/…)
