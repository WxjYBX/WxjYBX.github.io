    1. 主题下载(点击对应主题进入github在项目介绍中寻找下载方式，一般是 git clone )：https://hexo.io/themes/

    2. 打开配置文件：_config.yml

    3. 找到文件100行内容为：theme: 主题名

    4. 将./themes中的主题 文件夹的名字 代替掉配置文件中的 theme:主题名 中的主题名
